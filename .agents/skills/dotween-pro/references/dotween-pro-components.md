# DOTween Pro Components Overview

Core locations: Assets/Plugins/Demigiant/DOTweenPro

## Components
- DOTweenAnimation: inspector driven tween component. Primary tool.
- DOTweenPath: path based motion component using waypoints.
- DOTweenVisualManager: optional manager for preview controls in the editor.

## Extensions and extras
- DOTweenProShortcuts: adds DOSpiral shortcuts for Transform or Rigidbody.
- DOTweenTextMeshPro: adds TMP tween shortcuts (color, fade, text, per character).
- DOTweenTk2d: adds 2D Toolkit support (if tk2d is used).

## Module setup
- Use Tools > Demigiant > DOTween Utility Panel to enable modules (TMP, tk2d, etc).
- Some target types in DOTweenAnimation require the module to be enabled.
